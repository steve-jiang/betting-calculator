# Tote betting host

## How to run
- Install Node 8+ if your computuer doesn't have node 
- `git clone` this repo
- `cd` to the root of the repo
- run `node calculator`

## Format of the input
- You can add each bet individually or
- You can copy & past a list of bets in to the terminal
- The result input should be the last line input
- Hit `enter` after the result line then you will get the stdout of the dividents
